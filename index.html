HTTP1.1 vs HTTP2

Vignesh, June 15, 2022 #Task1

HTTP1.1:

HTTP1.1 was introduced in 1997. It supported connection reuse where there can be multiple requests and responses. It allowed to clients to request several resources from several servers at the same time which lead to some feasibility issues.

It indroduced warning headers to carry additional info about the status of message and also can define upto 24 status codes.

It used digest and NTLM authentication.

It provided faster delivery of web pages and reduces web traffic as compared to HTTP/1.0 .

HTTP2:

HTTP2 was introduced in 2015. It used multiplexing through streams which can be prioritized, can have dependencies and individual flow control. It also provides a feature called server push that allows the server to send data that the client will need but has not yet requested.

Underlying semantics of HTTP such as headers, status codes are same as in the previous version.

It is has better security while authentication compared to the previous versio and uses new TLS features to do the same.

HTTP/2 utilizes multiplexing and server push to effectively reduce the page load time by a greater margin along with being less sensitive to network delays.

Javascript objects and its internal representation

Vignesh, June 15, 2022 #Task1

Objects are a form of data type and forms building blocks of javascript. objects are not same as primitive data types (Number, string, Boolean,null..). objects contain a combination of primitive data types as well as reference data types and has properties associated with it. objects are denoted by curly braces {}.

objects basically created with key and value pairs.

Objects can be used with call by reference for better memory efficiency.

Example of object:

let bike = {name: 'Ninja 300', maker:'Kawasaki', engine:'300cc'}; where "name,maker,engine" represent keys and Ninja 300,Kawasaki,300cc represent the corresponding values.

There can be array of objects. The value of a particular key can be accessed by uisng dot(.) operator; bike.name (returns 'Ninja 300') .

We can also use arrays within objects (let bike = engine:[name: 'Ninja 300', maker:'Kawasaki','300cc', '200cc']) .
