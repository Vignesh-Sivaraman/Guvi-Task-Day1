Diff b/w Document & window objects

Vignesh, June 15, 2022 #Task1
Document objects

  1. Document object represents the loaded web page browser. By using document object, We can access the element in HTML page.
  2. The document object can be accessed by using "document.method_name" and "document.property_name".
  3. It is the object of window property and loaded inside the window.
  4. the document is part of both browser object model(BOM) and Document object model(DOM).
  5. All the tags, elements with attributes in HTML are part of the document.
  6. Example: document.title: will return the title of the document
Window objects
  1. It represents a browser window or frame that displays the contents of web page.
  2. The window object can be accessed by using "window.property_name".
  3. It is the very first object loaded in the browser.
  4. It is the object of the browser and only part of DOM not BOM.
  5. Global objects, functions, and variables of Javascript are members of window object.
  6. Example: window.inner height: will return the content area height of the browser.
